## Complai.nt | Unified AI Grievance Dashboard

**Complai.nt** is an intelligent communication hub designed to modernize how organizations handle customer feedback. Developed as a solution for the **iDEA 2.0 National Hackathon (Union Bank of India)**, this project bridges the gap between raw customer data and actionable insights.

> **🚧 Current Status: Work in Progress**
> The frontend UI and state management are fully built. We are currently developing the backend infrastructure to fully integrate the **Google Gemini API** for live multimodal data processing and automated AI responses. 

### 🚀 Key Features
* **Multimodal AI Analysis (WIP):** Integrating Google Gemini to process both text complaints and image-based evidence (receipts, screenshots).
* **Automated Escalation:** Smart categorization of complaints based on severity and SLA (Service Level Agreement) status.
* **Agent Copilot (WIP):** Generating AI-suggested responses and root-cause analysis to help support teams respond 40% faster.
* **Real-time Analytics:** A clean, responsive dashboard for tracking "Breached" vs. "Pending" tickets.

### 🛠️ Tech Stack
* **Frontend:** React.js, Vite, TypeScript
* **Styling:** Tailwind CSS (Modern, utility-first UI)
* **Backend/AI (In Progress):** Node.js, Gemini Multimodal Vision API
* **State Management:** React Hooks & Mock Data architecture
